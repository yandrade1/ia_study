# 🤖 AI Study Journey

Bem-vindo(a) ao meu repositório de estudos em **Inteligência Artificial, Machine Learning e Data Science**!  
Aqui registro minha evolução, anotações, códigos e experimentos práticos ao longo do caminho.

## 📌 Objetivo
Este repositório tem como propósito ser um **laboratório pessoal de aprendizado**.  
Nele vou praticar:
- Conceitos fundamentais de **álgebra linear e estatística** aplicados à IA
- Implementação de algoritmos básicos de Machine Learning
- Exploração de frameworks como **NumPy, Pandas, Scikit-learn, PyTorch/TensorFlow**
- Criação de mini-projetos para aplicar teoria na prática

## 📂 Estrutura
- `notebooks/` → Jupyter Notebooks com teoria + prática (cada semana/tema tem um arquivo próprio)  
- `data/` → Conjuntos de dados pequenos para experimentos  
- `projects/` → Projetos práticos pessoais (ex: análise de gastos, mini modelos de previsão, etc.)  
- `README.md` → Este arquivo que você está lendo agora 🚀  

## 🛠️ Tecnologias
- Python 3.x  
- NumPy / Pandas  
- Matplotlib / Seaborn  
- Scikit-learn  
- PyTorch (futuramente)  

## 📖 Recursos de Estudo
Estou seguindo uma trilha de aprendizado baseada em:
- Livro: *Agentic: Artificial Intelligence*
